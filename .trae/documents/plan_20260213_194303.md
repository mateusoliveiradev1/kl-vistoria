Entendido. Você quer um produto de classe mundial: **Design System UI/UX Impecável + Performance Máxima (100/100 Lighthouse)**.

Para atingir esse nível, não faremos apenas "código limpo", mas uma **Engenharia de Frontend Avançada**.

### **O Plano Definitivo: "KL Vistorias Masterpiece"**

#### **1. Arquitetura de Performance (Core Web Vitals)**
*   **Lazy Loading Inteligente**: Componentes pesados (como o Mapa do Google e Imagens grandes) só carregarão quando entrarem na viewport (`IntersectionObserver`). Isso garante um carregamento inicial instantâneo.
*   **Otimização de Imagens**: Usar tamanhos responsivos (`srcset`) e atributos `loading="lazy"` para evitar CLS (Cumulative Layout Shift) e LCP (Largest Contentful Paint) ruim.
*   **Minificação Extrema**: Otimizar o bundle final com tree-shaking agressivo (via Vite).

#### **2. Design System & UX (A Experiência do Usuário)**
*   **Micro-interações**: Botões com feedback tátil visual (`active:scale-95`), hover states suaves (`transition-all duration-300`), e focus rings acessíveis para navegação por teclado.
*   **Tipografia Escalar**: Usar `clamp()` para tamanhos de fonte fluidos que se ajustam pixel a pixel entre mobile e desktop, sem "pulos" bruscos.
*   **Contraste & Acessibilidade (a11y)**: Cores testadas para WCAG AA/AAA. Textos legíveis em qualquer fundo.
*   **Motion Design**: Animações de entrada (`framer-motion` like, mas leve com CSS puro) coordenadas para guiar o olhar do usuário (título -> subtítulo -> CTA).

#### **3. Componentização Atômica (Scalability)**
Vou criar uma biblioteca interna de componentes (`src/components/ui`):
*   `Container`: Para alinhar o conteúdo perfeitamente no centro.
*   `Section`: Para padronizar espaçamentos verticais (`py-16 md:py-24`).
*   `Button`: Com variantes (`primary`, `outline`, `ghost`) e tamanhos.
*   `Card`: Com efeitos de glassmorphism e hover states pré-definidos.
*   `Typography`: `Heading` e `Text` com pesos e cores padronizados.

#### **4. O Mapa (Correção & UX)**
*   **Endereço**: "Rua Estrada D, Quadra 1, Lote 11 - Luana Park".
*   **UX do Mapa**: Em vez de um iframe pesado carregando na hora, mostraremos um "placeholder" estático (imagem leve) que vira o mapa interativo **apenas quando o usuário clica ou chega perto**. Isso salva segundos preciosos de carregamento.

### **Execução Passo a Passo**
1.  **Fundação (`src/lib` & `src/data`)**: Configurar constantes globais e utilitários de classe (`cn` para merge de Tailwind).
2.  **Design Tokens (`tailwind.config` / CSS)**: Definir cores semânticas, sombras e bordas.
3.  **Componentes UI (`src/components/ui`)**: Construir os blocos básicos.
4.  **Refatoração das Seções**: Reescrever `Hero`, `Services`, `Features`, `Location` usando a nova arquitetura.
5.  **Otimização Final**: Implementar o Lazy Map e Lazy Images.
